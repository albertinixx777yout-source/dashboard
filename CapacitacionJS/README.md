# Capacitación JavaScript

## Unidad 1: Números
Se estudiaron los tipos numéricos, conversiones entre cadenas y números, uso de parseInt(), manejo de NaN, operador módulo y funciones matemáticas para redondeo.

## Unidad 2: Cadenas
Se aprendió a manipular cadenas mediante propiedades y métodos como length, toUpperCase(), toLowerCase(), substring() y plantillas literales.

## Unidad 3: Variables
Se revisó la declaración de variables con let y const, la reasignación de valores y el alcance de las variables.

## Unidad 4: Condicionales
Se implementaron estructuras de decisión utilizando if, else y el operador ternario para controlar el flujo de ejecución.

## Unidad 5: Funciones
Se desarrollaron funciones reutilizables utilizando parámetros y valores de retorno para organizar mejor el código.
