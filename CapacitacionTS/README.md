# Capacitación TypeScript

## Unidad 2: The any type
Se estudió el tipo especial any, que permite asignar cualquier valor a una variable y comprender sus ventajas y desventajas dentro de TypeScript.

## Unidad 3: TypeScript ESLint
Se aprendió a utilizar TypeScript ESLint para detectar errores comunes y mejorar la calidad del código mediante reglas de análisis estático.

## Unidad 4: Primitive types
Se revisaron los tipos primitivos de TypeScript como string, number, boolean, null y undefined, así como su declaración y uso.

## Unidad 5: Union types
Se estudiaron los tipos de unión utilizando el operador | para permitir que una variable acepte más de un tipo de dato.

## Unidad 6: The type alias
Se aprendió a crear alias de tipos mediante la palabra clave type para simplificar definiciones y mejorar la organización del código.
